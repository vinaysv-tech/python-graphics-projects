# Python Graphics Projects

A collection of Python programs that generate graphical patterns, visual art, and interactive games using `turtle` graphics and `tkinter`.

## 📁 Projects Included

### 🎮 Games

- **Tic Tac Toe (gameXO.py)** - A classic two-player Tic Tac Toe game built with `tkinter` featuring:
  - Interactive GUI with clickable buttons
  - Win detection for horizontal, vertical, and diagonal lines
  - Draw/tie detection
  - Restart functionality
  - Visual feedback with color highlighting

### 🎨 Graphical Patterns & Art

- **Heart Pattern (heart.py)** - Mathematical heart curve visualization using parametric equations
  - Uses trigonometric functions to generate the heart shape
  - Animated turtle graphics with color gradients

- **Tree Fractal (tree.py)** - Recursive fractal tree generation
  - Demonstrates recursive algorithms
  - Natural-looking branching patterns
  - Multi-colored visualization

- **Loop Graphics (loop.py)** - Geometric pattern created through iterative circle drawing
  - Rotational symmetry patterns
  - Layered circular designs

- **Numbered Scripts (1.py - 40.py)** - Various graphical pattern experiments including:
  - Spiral patterns
  - Geometric shapes
  - Mathematical curves
  - Abstract art generated through loops and angles

## 🛠️ Technologies Used

- **Python 3.x**
- **turtle** - For graphics rendering and pattern generation
- **tkinter** - For GUI-based game interface
- **math** - For trigonometric calculations

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system
- No additional packages required (uses standard library only)

### Running the Projects

1. Clone or download this repository
2. Navigate to the project directory
3. Run any script using Python:

```bash
# For graphical patterns
python heart.py
python tree.py
python loop.py
python 1.py

# For the game
python gameXO.py
```
